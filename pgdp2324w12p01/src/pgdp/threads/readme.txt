In den Packages 'start', 'race' und 'dead' findest sich jeweils die Lösung der entsprechenden Teilaufgabe.
D.h. der Code in 'start' löst Aufgabe 1, erzeugt aber eine Race Condition.
Der Code in 'race' löst diese Race Condition auf (was Aufgabe 2 war), erzeugt aber Deadlocks.
Der Code in 'dead' umgeht dann auch diese Deadlocks (was Aufgabe 3 war) und ist die endgültige Lösung dieser Aufgabe.
